# 💳 Day 3 – SaaS Pricing Cards UI

This project is part of my **Frontend Development Practice Series**, where I build UI components daily to strengthen my frontend fundamentals.

On **Day 3**, I practiced building a **responsive SaaS-style pricing section** using **HTML and CSS**, focusing on layout, visual hierarchy, and reusable UI components.

---

## 🎯 Objective

The goal of this practice was to learn how to structure and design a **pricing card component** commonly used in SaaS products.

Key areas of focus:

* Flexbox layout
* Card-based UI design
* Visual hierarchy
* Hover interactions
* Responsive layout thinking

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
* **Flexbox**
* **Remix Icon** for feature icons
* **Google Fonts (Inter)** for typography

---

## ✨ Features

* Three responsive pricing cards
* Highlighted **featured plan**
* Hover animation for cards
* Pill-style badges for plan names
* Feature list with check icons
* Clean SaaS-style UI design
